# Viewing slides from a previous talk

*NOTE*: You'll need to have Docker and Docker Compose installed for this to work.

Run `scripts/launch_slides.sh [talk_name]` where `[talk_name]` is the name of the
talk.

## Examples

```sh
$: scripts/launch_slides.sh all-day-devops-2019       # Shows slides from ADDO 2019
$: scripts/launch_slides.sh "All Day DevOps 2019"     # Same as above.
$: scripts/launch_slides.sh "aLl DaY dEvOpS 2019"     # Same as above, but plz no
```
